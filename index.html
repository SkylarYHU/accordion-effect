• A variable is a storage location in programming that holds a value. It allows developers to store, retrieve, and manipulate data throughout their code. Variables can store different types of data, such as numbers, strings, or objects.

  • A function is a reusable block of code designed to perform a specific task. Functions help organize code, improve readability, and make it easier to maintain. They can accept inputs (parameters) and return an output.

  • An array is a data structure that can store a collection of elements, usually of the same data type, in a single variable. Arrays are useful for organizing and managing lists of items.

  • An object is a complex data structure that can store multiple values in the form of properties (key-value pairs). Objects are fundamental in object-oriented programming and are used to represent real-world entities.

  • A loop is a programming construct that repeats a block of code as long as a specified condition is met. Loops are useful for automating repetitive tasks and processing collections of data.
